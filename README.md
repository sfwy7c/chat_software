基于 Java Socket 的简易聊天室 (C/S 架构)

>  **这是一个用于学习 Java 网络编程、多线程并发和 Swing GUI 开发的练习项目。** 通过构建一个多人在线聊天室，深入理解 C/S（客户端/服务端）通信模型。

##  功能特点

-  **图形化界面**：基于 Swing/AWT 实现，操作直观。
-  **多人同时在线**：服务端支持多个客户端同时连接并进行消息转发。
-  **实时群聊**：任意客户端发送的消息，会被服务器广播给当前所有在线用户。
-  **并发处理**：服务端为每个连接的客户端开辟独立线程，保证消息处理的及时性。
-  **优雅的连接管理**：客户端可主动断开连接，服务端能感知并剔除离线用户。

## 🛠️ 技术栈

| 模块 | 技术选型 |
| :--- | :--- |
| 编程语言 | Java (JDK 8+) |
| 网络通信 | `java.net.Socket` / `ServerSocket` (TCP 协议) |

| 用户界面 | `javax.swing` / `java.awt` (GUI) |

## 快速开始（本地运行）

请确保你的本地环境已安装 JDK 8 或更高版本，并配置好了环境变量。

### 第一步：启动服务端 (Server)

1. 进入项目根目录下的 `Server/src` 文件夹。
2. 找到并运行 `ServerStart.java`（或主启动类）。
3. 观察控制台输出：当出现 **“服务器已启动，等待客户端连接...”** 提示时，表示服务端已就绪。

### 第二步：启动客户端 (Client)

1. 打开一个新的终端或 IDE 窗口，进入 `Client/src` 文件夹。
2. 找到并运行 `ClientStart.java`（或登录窗口类）。
3. **模拟多人聊天**：你可以重复第 2 步，在本地同时打开多个客户端窗口（系统会自动分配不同端口），实现自己跟自己聊天的测试效果。

> 注意**：客户端连接服务端时，请确保连接的 IP 地址填写正确。如果在本机测试，默认使用 `127.0.0.1` 或 `localhost`。

## 使用说明

1. **连接**：启动客户端后，输入你的昵称（可选功能，视具体实现而定）点击“连接/登录”。
2. **发送消息**：在输入框中输入内容，点击“发送”按钮或按回车键，消息将出现在聊天记录区域。
3. **退出**：关闭客户端窗口即可断开连接（或通过界面上的“退出”按钮）。

##  项目涉及的核心知识点

- **Socket 编程**：理解三次握手与 TCP 可靠传输。
- **BIO 模型**：阻塞 I/O 下线程池与 `while(true)` 循环的经典写法。
- **对象流传输**：将 Java 对象（如 `Message` 类）通过网络直接传输，避免繁琐的字符串解析。
- **GUI 线程安全**：理解 Swing 的 `EventQueue.invokeLater()` 用于更新 UI，避免界面卡死。
